Welcome to Calm Thinking

In this study, we are going to focus on how to remain calm in situations that typically cause you anxiety.

You will complete a computer program that will describe different situations and encourage you to think about how those situations will turn out. This computer program is new but is based on similar programs that have shown positive effects in changing future thinking.

Because the program is still being tested, we have people trying different versions of it to see which version works best — we think some of these computer programs will lead to a healthier evaluation of the future.

Let's get started!
